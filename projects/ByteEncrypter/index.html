Byte Encrypter

This is a simple C program that I made because manually encrypting bytes was very tedious (especially when there's a lot of them) This took about 5 hours of trial and error to make it work.

It's still in kinda shitty state, I'll probably improve it once I make a program to decrypt those bytes.

How It works?

This software asks for a string of bytes and which bits do you wan't to flip.
After that it shortens each byte from 8 to 7 bits shaving off the first 0 in each byte.
Example:
01111000 00110011 --> 1111000 0110011

Next it flips the bits which you specified to flip (Flipping the 8th bit doesn't change anything since it gets removed, but we can use it to confuse people who don't know how it works).
Example:
flip bits 7 2 (the order doesn't matter)
01100110 01110101 -->0100100 0110111

In order to make it possible to decrypt the software outputs 7~X_
(7~ because each byte has 7 bits now and X stands for bits you specified to flip) in front of the string.
Example:
7~327_00010000101111010010101000111100111

Now you can try to decrypt this message :)
